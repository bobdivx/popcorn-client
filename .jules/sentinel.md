## 2025-03-13 - [Sentinel] Fix XSS Vulnerability in DescriptionPreview
**Vulnerability:** A Cross-Site Scripting (XSS) vulnerability was found in `src/components/upload/DescriptionPreview.tsx`. User-provided content (either `html` or `raw` BBCode) was rendered using `dangerouslySetInnerHTML` without proper sanitization. The previous sanitization only removed `{}` placeholders but did not sanitize malicious tags or attributes.
**Learning:** React/Preact's `dangerouslySetInnerHTML` is extremely risky without robust external sanitizers like `DOMPurify` or `isomorphic-dompurify`. Custom regex string manipulation (like `.replace(/\{\}/g, '')`) is not sufficient for XSS protection. Because this is an Astro/Preact frontend, `isomorphic-dompurify` must be used to ensure sanitization occurs properly during SSR to avoid hydration mismatches.
**Prevention:** Always use `isomorphic-dompurify` when rendering untrusted HTML from backends/APIs or directly from user input via `dangerouslySetInnerHTML` in Astro/Preact applications.
